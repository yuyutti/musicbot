## 【versions】
Node.js v18.15.0  
npm v9.5.0  
package package.jsonに記載
Mysql version知らない

### 【使い方】
1.env.exampleを編集して必要事項を入力してください

[DiscordBOTのtoken入手先](https://discord.dev)  

2 ```npm i```で必要packageをダウンロード

3 ```node index.js```で起動！
